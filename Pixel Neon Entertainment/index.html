Pixel Neon Entertainment — Bağımsız Dijital Stüdyo

Pixel Neon Entertainment, 2026 yılında yazılım geliştirici Görkem Akkaya tarafından kurulan Türkiye merkezli bağımsız bir dijital stüdyodur. Stüdyo; oyun geliştirme, mobil uygulama, web geliştirme ve Windows masaüstü yazılımları alanlarında hizmet vermektedir.

Hizmetler

  • Oyun Geliştirme: Unity ve Unreal Engine kullanılarak 2D ve 3D oyun geliştirme. Prototipten yayına kadar tam kapsamlı üretim.
  • Mobil Uygulama Geliştirme: React Native ile iOS ve Android platformları için çok platformlu mobil uygulama.
  • Web Geliştirme: HTML5, CSS3 ve JavaScript ile etkileşimli, modern web siteleri ve uygulamalar.
  • Masaüstü Yazılım Geliştirme: C# ve WPF/.NET ile Windows platformu için masaüstü uygulamaları.
  • UI/UX Tasarım: Fütüristik ve kullanıcı dostu arayüz tasarımı.

Tamamlanan Projeler

  • P Downloader — Windows için cyberpunk temalı gelişmiş indirme yöneticisi. HTTP, FTP, Torrent ve Magnet bağlantı desteği, çoklu parça indirme, ultra hız modu, SQLite geçmişi, Google Drive ve OneDrive entegrasyonu, şifreli ZIP/RAR/7Z açma, 6 neon tema ve Türkçe/İngilizce/Almanca dil desteği sunar. C#, WPF ve .NET 10 ile geliştirilmiştir. Ücretsiz olarak indirilebilir.
  • Görsel Yönetici — Windows için toplu resim yönetimi ve görsel işleme masaüstü uygulaması. Hızlı dosya taşıma, düzenleme, hafif ve kullanıcı dostu arayüz özellikleri sunar. Ücretsizdir.
  • Pixel Neon Web Sitesi — Saf HTML, CSS ve JavaScript ile yazılmış cyberpunk temalı portfolyo sitesi. PWA desteği, çevrimdışı çalışma, Türkçe/İngilizce dil seçeneği içerir.

Kullanılan Teknolojiler

Unity, Unreal Engine, C#, WPF, .NET 10, React Native, JavaScript, HTML5, CSS3, Python, SQL, Node.js, Docker, AWS, REST API, OAuth.
