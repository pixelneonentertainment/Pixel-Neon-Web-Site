Pixel Neon Entertainment — Official Website

"Every pixel is placed with purpose."

The official portfolio website of Pixel Neon Entertainment, an independent digital studio founded by Görkem Akkaya in 2026. Built entirely from scratch using pure HTML, CSS, and JavaScript — no frontend frameworks, no build tools, no bloat.

About the Studio
Pixel Neon Entertainment is a solo indie studio based in Turkey, focused on creating aesthetic and functional digital products across multiple platforms:

🎮 Game Development — 2D & 3D games with Unity and Unreal Engine
📱 Mobile Development — Cross-platform apps with React Native (iOS & Android)
🌐 Web Development — Interactive web experiences with HTML/CSS/JS
🖥️ Desktop Software — Windows applications with C# and WPF / .NET
